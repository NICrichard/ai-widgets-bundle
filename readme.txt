This plugin adds some Bootstrap widgets for use with SiteOrigin's SiteBuilder plugin.

== Installation ==

1. Upload `ai-widgets-bundle` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Requirements ==

It is necessary to have installed, and activated, SiteOrigin's SiteBuilder/PageBuilder plugin before installing and activating this one.
